Página Romântica - estrutura organizada

Arquivos principais:
- index.html: página principal pronta para GitHub Pages
- dengo.mp3: música da página
- dados.js: mapa das imagens, mantendo referência ao nome original
- img/: fotos padronizadas como memoria001.jpg, memoria002.jpg, ...

Para publicar:
1. Suba todo o conteúdo desta pasta no repositório do GitHub Pages.
2. Mantenha a pasta img junto do index.html.
3. Para trocar uma foto no futuro, substitua o arquivo correspondente mantendo o mesmo nome, exemplo: img/memoria010.jpg.
